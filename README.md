# Notes App 

A simple console-based notes application written in Python.

## Features

- Add new notes  
- View all notes  
- Count total notes  
- Delete notes  
- Edit existing notes  
- Search notes by keyword  

##  What I learned

- Working with files in Python  
- Reading and writing data (`.txt`)  
- Structuring code using functions  
- Building a simple interactive menu  
- Handling user input and errors  

## How to run

1. Make sure you have Python installed  
2. Run the script:

```bash
python main.py
